# Accordion React Project

## Description

This is a simple **Accordion component** built with **React**, without any library where the content is dynamically loaded from a `data.js` file.  
The project demonstrates:

- React functional components
- State management with `useState`
- Mapping through data arrays to render components dynamically
- Simple UI interactivity

## Features

- Expand/collapse sections individually
- Dynamic rendering from a JavaScript data file
- Clean and minimal design
- Easy to extend with more sections

## Technologies Used

- React (Functional Components)
- JavaScript (ES6)
- TailwindCSS for styling
